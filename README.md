# Keystone Project Starter

Run

```
yarn dev
```

## Env file

Need to create .env file

```
S3_BUCKET_NAME=''
S3_REGION=''
S3_ACCESS_KEY_ID=''
S3_SECRET_ACCESS_KEY=''
COOKIE_SECRET=""
DATABASE_URL= ""
STRIPE_SECRET= ""
MAIL_HOST= ""
MAIL_PORT= ""
MAIL_USER= ""
MAIL_PASS= ""
FRONTEND_URL= "http://localhost:3000"
ASSET_BASE_URL= 'http://localhost:3008'
BACKEND_URL="http://localhost:3008/api/graphql"
NODE_ENV="development"
```
